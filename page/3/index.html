Quick BI简易操作说明

0. 产品介绍

Quick BI 是一个基于云计算的灵活的轻量级的自助 BI 工具服务平台。

登陆网址：https://data.aliyun.com/product/bi

文档网址：https://www.alibabacloud.com/help/zh/product/30343.htm

视频网址：https://edu.aliyun.com/course/40

1. 添加数据源

在Quick BI 产品详情页 ，单击管理控制台，就可以进入Quick BI控制台。进入Quick BI界面，点击工作空间-> 数据源->新增，添加数据，如图1.1所示

图1.1


图1.2

如图1.2所示，提供了三种不同方式

  • 来自云数据库–云上的数据库。官网推荐MaxCompute，然而要收费
  • 来自自建数据库–连接自己服务器上的数据库
  • 来自其他–来自本地

2. 将数据源添加到数据集

连接到你的数据库，它会展示你数据库所有的表，可以选择某一张或某几张表来使用

如图2.1 选择数据库中的一张表作为数据集

如图2.2 点击数据集->我的数据集查看数据

6. 常见问题

    | 1 | 2 | 3 |
    | | | |
